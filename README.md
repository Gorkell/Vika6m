# Магическая Комната - Романтический Сайт-Подарок

Интерактивный сайт-квест для дня рождения, созданный с любовью ❤️

## 🌟 Особенности

- **4 тематические комнаты** с уникальным контентом
- **Интерактивные замочки** с загадками и диалогами
- **Система прогресса** с сохранением в localStorage
- **Магическая атмосфера** с анимациями и эффектами
- **Пасхалка** для самых внимательных
- **Аудио сообщения** для каждой секции

## 📁 Структура проекта

```

```

## 🎮 Как работает

### Главная страница
- Отображает доступные комнаты
- Показывает прогресс прохождения
- Финальная комната открывается после прохождения всех остальных

### Комнаты
Каждая комната содержит 3 текстовых блока с:
- Текстом (заглушка "Тут будет текст")
- Летающим замочком с загадкой
- Аудиоплеером для голосового сообщения

### Замочки
- **Комната воспоминаний**: Загадка про имя с множеством вариантов ответов
- **Комната комплиментов**: Любой осмысленный ответ принимается
- **Комната забавных моментов**: Любой ответ принимается с юмором

### Система ответов (Комната воспоминаний)

**✅ Правильные ответы:**
- `Вика` → "Верно! Но это лишь одно из имён Богини!"
- `Викачка` / `ВИКАЧКА` → "Женя что ли решил ответить? Ну ладно, это правильно!"
- `Викусик` / `викусик` → "Абсолютно верно! 🥳"
- `Виктория` → "Фигасе, с паспорта смотришь? Ну да, это правильный ответ!"
- `Дроздоперма` / `дроздоперма` → "Викуся, такое могла написать только ты 😅"

**❌ Неправильные ответы:**
- `вика` / `виктория` → "Имя Богини с маленькой буквы!? Переписывай! 😡"
- `Редмур` / `Левинская` → "Это фамилия Богини, а не имя 😒"
- `редмур` / `левинская` → "Фамилия и ещё с маленькой буквы!"
- `Мопс` / `мопс` → "Мне похуй, я мопс! Переписывай"

## 🌐 Размещение на GitHub Pages

1. Загрузите все файлы в репозиторий GitHub
2. В настройках репозитория → Pages выберите ветку и папку
3. Сайт будет доступен по адресу: `https://username.github.io/repository-name`

## 🎨 Кастомизация

### Изменение текстов
Замените `"Тут будет текст"` в HTML файлах на ваши тексты.

### Изменение загадок
В файле `script.js` в функции `getRiddleData()` можно изменить:
- Вопросы загадок
- Варианты правильных и неправильных ответов
- Текст ответов

### Изменение цветов
В файле `style.css` измените градиенты в:
- `body` - основной фон
- `.riddle-content` - фон модального окна
- `.progress-fill` - цвет прогресс-бара

## 🐣 Пасхалка

Кликните 5 раз по фону страницы, чтобы увидеть секретное сообщение!

## 💾 Сохранение прогресса

Прогресс автоматически сохраняется в localStorage браузера:
- Состояние замочков
- Пройденные комнаты
- Общий процент завершения

- JavaScript включен
